# Procesos para instalacion por Termux

```bash
> pkg install git
> pkg install bash
> pkg install nodejs-lts
> pkg install ffmpeg
> git clone https://github.com/ThisIsInky/MamiChan
> cd MamiChan
> bash install.sh
> npm start
```
